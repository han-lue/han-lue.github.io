Deluxe 220 - Digital Typewriter Font

Deluxe 220 is a monospaced font inspired by the typeface of the Brother Deluxe 220 typewriter from the 1970s.

Although it appears as a single typeface, it's actually built from four separate font variations. A JavaScript function randomly mixes these variations as the text renders, creating the subtle imperfections of typewritten text.

The font files and code snippet are available at github.com/han-lue/deluxe220.
